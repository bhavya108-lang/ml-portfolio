# ML Portfolio — Andrew Ng's Machine Learning Specialization

Projects built in lockstep with each week of Course 1 (Supervised Machine Learning: Regression and Classification). Each project is scoped strictly to concepts taught up to that point in the course — no forward-looking material.

## Projects

| Week | Project | Concepts |
|------|---------|----------|
| 1 | [Salary Prediction](./week1-salary-prediction) | Simple linear regression, cost function, gradient descent (from scratch, NumPy) |
| 2 | [House Price Prediction](./week2-house-price-prediction) | Multiple linear regression, vectorization, feature scaling, feature engineering, polynomial regression |

## Stack
Python, NumPy, Matplotlib, Jupyter — models implemented from scratch (no scikit-learn) to demonstrate understanding of the underlying math.
